jpg

Best compression for pictures

Use it for photography

png

Supports alpha channel

Use it when an image has transparency

gif

Limited color palette of 256 color

Has animation support

Use it for small animated images

svg

XML vector image format

Use it for icons or vector images
  • It's accessible (has description and title tags)
  • It can be handled by CSS and Javascript
  • It's readable
  • You can write it!
